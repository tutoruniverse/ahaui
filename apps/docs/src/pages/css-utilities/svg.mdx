# SVG

CSS Utilities for SVG

## Fill

Use the <code>.u-fillCurrent</code> class to set the fill color of an SVG to the current text color, or use the <code>.u-fillNone</code> class to set its fill color to transparent.

<div className="u-positionRelative u-border u-borderUltraLight u-paddingMedium u-marginBottomLarge">
  <div class="u-marginBottomMedium">
    <Icon size="extraLarge" name="thumbsUp" className="u-textPositive u-fillCurrent" />
    <div><code>.u-textPositive .u-fillCurrent</code></div>
  </div>
  <div class="u-marginBottomMedium">
    <Icon size="extraLarge" name="thumbsUp" className="u-textPrimary u-fillCurrent" />
    <div><code>.u-textPrimary .u-fillCurrent</code></div>
  </div>
  <div class="u-marginBottomMedium">
    <span class="u-inlineBlock u-border u-borderDashed">
      <Icon size="extraLarge" name="thumbsUp" className="u-fillNone" />
    </span>
    <div><code>.u-fillNone</code></div>
  </div>
</div>

## Stroke

Use the <code>.u-strokeCurrent</code> class to set the stroke color of an SVG to the current text color. Also add the <code>.u-fillNone</code> to remove the fill color.

<div className="u-positionRelative u-border u-borderUltraLight u-paddingMedium u-marginBottomLarge">
  <div class="u-marginBottomMedium">
    <Icon size="extraLarge" name="thumbsUp" className="u-textPrimary u-strokeCurrent u-fillNone" />
    <div><code>.u-textPrimary .u-fillNone .u-strokeCurrent</code></div>
  </div>
  <div class="u-marginBottomMedium">
    <Icon size="extraLarge" name="thumbsUp" className="u-textPrimary u-strokeCurrent u-stroke-2 u-fillNone" />
    <div><code>.u-textPrimary .u-fillNone .u-strokeCurrent .u-stroke-2</code></div>
  </div>
</div>
