# Grid

Using a grid system is very helpful when designing web pages. It makes it easier to place elements on the page.

## Introduction
The grid system normally has 12 columns (12-column grid system) and has a total width of 100%.

It can also be transformed to other grid systems for the different purposes and layouts, including: 2-column, 3-column, 4-column, 5-column, 6-column, 8-column or 10-column system.

## Grid Elements

### Row

A row is the horizontal space which takes 100% of the width, we use Grid to name a row.

Gutter options:

* `Grid`  - has gutter between columns by default
* `Grid--smallGutter`  - small gutter
* `Grid--withoutGutter`  - no gutter

### Column

Column is where to place content. Columns should be placed directly in row. The maximum number of columns in a row is 12. Columns are used withu-size*  in CSS utilities.

#### Sizes

* `u-sizeFit`  - Make an element shrink wrap its content with flex-basis.
* `u-sizeFull`  - Make an element the width of its parent.
* `u-sizeFill`  - Make an element fill the remaining space. Distribute space evenly on multiple elements.
* `u-sizeFillAlt`  - An alternative method to make an element fill the remaining space. Distribute space based on element width.
* `u-sizeXofY`  (numerous) - Specify the proportional width of an object.

`X`  must be an integer less than  `Y`.

`Y`  can be any of the following numbers: 2, 3, 4, 5, 6, 8, 10, 12.

#### Responsive classes

The grid includes six tiers of predefined classes for building complex responsive layouts. Customize the size of your columns on extra small, small, medium, large, or extra large devices however you see fit.

* `sm:u-sizeXofY`  - To use at the smallest Media Query breakpoint.
* `md:u-sizeXofY`  - To use at the medium Media Query breakpoint.
* `lg:u-sizeXofY`  - To use at the large Media Query breakpoint.
* `xl:u-sizeXofY`  - To use at the extra large or desktop Media Query breakpoint.
* `xxl:u-sizeXofY`  - To use at the monitor Media Query breakpoint.

### Offsetting columns

#### Offset classes

* `u-offsetXofY`  (numerous) - Specify the proportional offset before an object.

#### Responsive classes

The offset includes six tiers of predefined classes for building complex responsive layouts. Customize the size of your columns on extra small, small, medium, large, or extra large devices however you see fit.

* `sm:u-offsetXofY`  - To use at the smallest Media Query breakpoint.
* `md:u-offsetXofY`  - To use at the medium Media Query breakpoint.
* `lg:u-offsetXofY`  - To use at the large Media Query breakpoint.
* `xl:u-offsetXofY`  - To use at the extra large or desktop Media Query breakpoint.
* `xxl:u-offsetXofY`  - To use at the monitor Media Query breakpoint.
